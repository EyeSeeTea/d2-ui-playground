## Setup

```
$ yarn install
```

## @dhis2/d2-ui-interpretations

Development:

In d2-ui folder:

```
$ yarn install
$ cd packages/interpretations
$ yarn link
$ yarn watch
```

In d2-ui-playground folder:

```
$ yarn link "@dhis2/d2-ui-interpretations"
$ yarn start
```
